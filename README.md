# 📊 Census of India Dashboard (Power BI)

## 📌 Project Overview

This project analyzes Census of India data using Power BI to understand population distribution, gender ratio, literacy rates, and workforce participation.

## 🎯 Objectives

* State-wise population analysis
* Gender distribution comparison
* Literacy rate insights
* Workforce participation trends

## 🛠️ Tools Used

* Power BI
* Power Query
* DAX

## 📈 Key Insights

* Population distribution is uneven across states
* Some states show gender imbalance
* Literacy rates vary significantly
